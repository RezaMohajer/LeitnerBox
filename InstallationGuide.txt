1- First install sadaf framework (github.com/milanifard/sadaf)
2- copy this module to same folder as sadaf www
3- Rename sadaf's www dir to sadaf
4- Import tables.sql by running script in your mysql
5- Copy BoxView.php and  LeitnerBox.php to sadaf/sadaf/ folder (the same folder as main.php)
6- Edit sadaf's htaccess and add model and config directories like this according to your server folder: ;C:\wamp64\www\LeitnerBox\config;C:\wamp64\www\LeitnerBox\model
7-  Open /LeitnerBox.php inside sadaf directory containing main.php and enjoy
